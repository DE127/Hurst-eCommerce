========NẾU CÁC BẠN THẤY HAY CÓ THỂ CHO MÌNH 1 START VÀ 1 FOLLOW NHÉ!========================
Project này được viết bằng PHP, MySQL, HTML, CSS, JS, Jquery, Bootstrap, Fontawesome, ...

## Cài đặt
1. Clone project về máy copy tất cả các file vào httpdocs hoặc public_html trong xampp hoặc wampp,...
2. Tạo database có tên là `hurst_ecm` trong phpMyAdmin và import file `hurst_ecm.sql` vào database đó
3. Vào file `config/config.php` để cấu hình database của bạn mặc định đã cấu hình đúng cho localhost
